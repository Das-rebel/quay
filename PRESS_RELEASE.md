## Das-rebel Launches Quay — The Open-Source AI Software Factory

*Self-hosted autonomous agents with MCP support, real-time dashboards, and per-call cost tracking*

---

**July 2026** — Das-rebel today announced the launch of **Quay**, an open-source AI Software Factory that enables teams to deploy autonomous coding agents without vendor lock-in or opaque pricing.

Quay fills the gap between research-grade AI frameworks (like AutoGen, CrewAI) that require months of integration work, and enterprise platforms (like ServiceNow AI Agents, Microsoft Copilot Studio) that are cloud-only with proprietary LLMs and enterprise price tags.

### What is Quay?

Quay is an MIT-licensed autonomous AI agent platform that runs entirely on your infrastructure. It provides:

- **Autonomous Agent Pipelines** — Role-based agents (Engineer → Reviewer → Security → Deployer) with customizable stages
- **MCP Native Support** — First-class Model Context Protocol integration for connecting any MCP tool server
- **Mission Control Dashboard** — Real-time dark UI with KPI cards, 7-state Kanban board, and SSE streaming
- **Cost Transparency** — Every LLM call logged with token counts and USD cost
- **SQLite Audit Trail** — Complete schema for compliance, logging every state transition and tool call
- **A3M Router Integration** — Route through 40+ LLM providers (OpenAI, Anthropic, Groq, Ollama, and more)
