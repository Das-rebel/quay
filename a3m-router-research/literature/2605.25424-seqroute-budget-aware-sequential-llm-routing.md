# SeqRoute: Global Budget-Aware Sequential LLM Routing via Offline Reinforcement Learning

## Paper Details
- **arXiv ID**: 2605.25424
- **Authors**: Zhongling Xu, Shunan Zheng, Wei Wang
- **Year**: 2026
- **Categories**: cs.LG, cs.AI
- **URL**: https://arxiv.org/abs/2605.25424

## Key Innovation
Sequential LLM routing with global budget awareness using offline RL. Unlike single-step routing, considers cumulative cost and performance across multiple turns. Uses offline reinforcement learning to learn a routing policy that optimizes long-term utility under budget constraints.

## Relevance to A3M Router Improvement
- **High relevance** - Budget-aware routing is key for cost optimization
- Could enhance A3M Router's cost control mechanisms
- Offline RL approach may be more practical than online learning
