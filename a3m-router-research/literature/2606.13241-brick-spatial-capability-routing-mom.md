# Brick: Spatial Capability Routing for the Mixture-of-Models (MoM) Paradigm

## Paper Details
- **arXiv ID**: 2606.13241
- **Authors**: Francesco Massa, Marco Cristofanilli
- **Year**: 2026
- **Categories**: cs.AI
- **URL**: https://arxiv.org/abs/2606.13241

## Key Innovation
Defines query difficulty as the key challenge in LLM routing. Instead of surface features (domain labels, keywords, token count), uses within-domain variance that actually determines model success. Frontier models cost 10-100x more than local open-weight models, so accurate routing is critical at production scale.

## Relevance to A3M Router Improvement
- **High relevance** - Query difficulty estimation is core to routing decisions
- Could improve A3M Router's ability to assess query complexity
- May reduce costs by better matching queries to model capabilities

## Notes
Technical report, 17 pages. Addresses the fundamental problem of defining and measuring query difficulty.
