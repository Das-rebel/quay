# LLM Routing with Dueling Feedback

## Paper Details
- **arXiv ID**: 2510.00841
- **Authors**: Chao-Kai Chiang, Takashi Ishida, Masashi Sugiyama
- **Year**: 2025
- **Categories**: cs.LG
- **URL**: https://arxiv.org/abs/2510.00841

## Key Innovation
Uses dueling feedback mechanism for LLM routing - comparing pairs of models on the same query to learn relative performance. More sample-efficient than learning absolute scores.

## Relevance to A3M Router Improvement
- **Medium-high relevance** - Could improve router training efficiency
- May reduce the number of queries needed to train the router

## Notes
October 2025 paper. Focuses on training methodology.
