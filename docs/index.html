Features

Everything an AI engineering team needs

From task creation to deployment — all tracked, logged, and observable in real time.

🤖

Role-Based Agent Pipelines

Engineer → Reviewer → Security → Deployer pipeline. Swap or extend any stage. Each stage is a separate LLM call with full tool access.

🔌

MCP Tool Integration

Connect any Model Context Protocol tool server. Filesystem, git, Slack, Docker — if there's an MCP server, Quay can use it.

💰

Cost Transparency

Every LLM call logged with tokens and cost in USD. Per-task, per-project, and global dashboards. No surprise bills.

🛡️

Self-Hosted & Open

Run entirely on your own infrastructure. Bearer-token auth, SQLite audit trail, no vendor lock-in. Your code, your infra.

📊

Mission Control Dashboard

Real-time dark UI with KPI cards, Kanban board, and live activity feed. SSE streaming — updates the instant state changes.

🧠

Three-Tier Agent Memory

Short-term (30-min cache per run), medium-term (7-day lessons per project), long-term (agent capability graph with success rates).

🔄

SWE-Agent Loop

Observe → Plan → Act → Reflect. Each cycle is a separate step_attempt row in SQLite with full trace for debugging.

📡

Real-Time SSE Streaming

Server-Sent Events for every task lifecycle event. Zero polling. Task creation, state transitions, pipeline completions — all live.

🔀

A3M Multi-Provider Routing

Route tasks to the optimal LLM provider per stage. OpenAI, Anthropic, Groq, Ollama, and 40+ more — adaptive routing built in.

Quick Start

Up and running in 60 seconds

No external dependencies beyond Bun. SQLite is built in.

bash 
git clone https://github.com/Das-rebel/quay.git && cd quay
bun install
bun run scripts/seed.ts      # seed demo data
bun run src/server/index.ts    # API on http://localhost:3001
npm run dev                    # Dashboard on http://localhost:5173
bash 
# Configure MCP tool servers (optional)
export QUAY_MCP_SERVERS='[{"name":"filesystem","command":"npx","args":["-y","@modelcontextprotocol/server-filesystem","/tmp"]}]'

# Set a secure API key
export QUAY_API_KEY="your-secret-key-here"

# Restart server
bun run src/server/index.ts
